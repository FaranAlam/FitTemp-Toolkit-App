# 📱 FitTemp Toolkit - Android App

Welcome to **FitTemp Toolkit**, a natively developed Android application designed for quick daily health and utility calculations.

### 🚀 Features
* **BMI Calculator:** Calculates Body Mass Index accurately by taking inputs in Kilograms, Feet, and Inches.
* **Temperature Converter:** Instantly converts Fahrenheit to Celsius with precision formatting.
* **Clean UI/UX:** Built with a professional, card-based responsive design.

### 🛠️ Tech Stack
* **Language:** Kotlin / XML
* **IDE:** Android Studio
* **Architecture:** Standard Android MVC
